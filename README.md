## Claude Code

You can register this repository as a Claude Code Plugin marketplace by running the following command in Claude Code:

```
/plugin marketplace add mcollina/skills
```

Then, to install a skill:
1. Select `Browse and install plugins`
2. Select `matteo-collina-skills`
3. Select the skill you want (e.g. `node`, `fastify`, `typescript-magician`)
4. Select `Install now`

Alternatively, directly install a skill via:

```
/plugin install node@matteo-collina-skills
/plugin install fastify@matteo-collina-skills
/plugin install typescript-magician@matteo-collina-skills
```

After installing, you can use the skill by just mentioning it. For instance, if you install the `node` plugin, you can ask Claude Code to help you with Node.js best practices and it will automatically use the skill.
